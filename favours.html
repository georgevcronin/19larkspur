Favour points ledgerİyilik puanı defteri

Log favours done between housemates and keep score. Doing a favour earns you the points; receiving one costs you them.Ev arkadaşları arasında yapılan iyilikleri kaydedin ve skoru tutun. İyilik yapmak puan kazandırır; iyilik almak puan kaybettirir.

Cloud sync is on. Entries save to Firestore and update live on every housemate's device. Use Export CSV for backups.Bulut eşitleme açık. Kayıtlar Firestore'a kaydedilir ve her ev arkadaşının cihazında anında güncellenir. Yedek için CSV dışa aktarı kullanın.

Cloud sync is not set up yet, so entries currently save in this browser only. Use Export CSV to share or back up the ledger.Bulut eşitleme henüz kurulmadı; kayıtlar şimdilik yalnızca bu tarayıcıda saklanıyor. Defteri paylaşmak veya yedeklemek için CSV dışa aktarı kullanın.

How to turn on cloud sync (Firestore)Bulut eşitleme nasıl açılır (Firestore)

One housemate does this once:Bir ev arkadaşı bunu bir kez yapar:

  1. Create a project at console.firebase.google.com.console.firebase.google.com adresinde bir proje oluşturun.
  2. Add a Firestore database and publish the access rules shown in firebase-config.js.Bir Firestore veritabanı ekleyin ve firebase-config.js içinde gösterilen erişim kurallarını yayımlayın.
  3. Register a Web app, copy its config values into firebase-config.js, and push.Bir Web uygulaması kaydedin, yapılandırma değerlerini firebase-config.js dosyasına kopyalayın ve gönderin.

The page detects the config automatically — no other changes needed.Sayfa yapılandırmayı otomatik algılar — başka değişiklik gerekmez.
